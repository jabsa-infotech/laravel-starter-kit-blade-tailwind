## Uses

### Configure ports in your env like this

Helps when you have already running something on default ports

``` .env
APP_PORT=80 # replace with port number
FORWARD_DB_PORT=3306
FORWARD_PHPMYADMIN_PORT=8081
VITE_PORT=5174
```
